📊 Vrinda Store Sales Analysis Dashboard

End-to-End Data Analytics Project | Excel • Data Visualization

This project analyzes Vrinda Store’s 2022 sales data to uncover insights into customer behavior, sales performance, and regional trends. The goal is to support data-driven business decisions using an interactive dashboard.

📌 Project Overview
The dashboard transforms raw sales data into meaningful insights:
Customer Analysis: Gender and age-wise contribution
Sales Channels: Performance across Myntra, Flipkart, etc.
Order Insights: Delivered, Cancelled, and Returned orders
Regional Performance: State-wise sales distribution
Trend Analysis: Monthly sales and order patterns

🛠️ Tech Stack & Tools
Microsoft Excel
Pivot Tables
Charts & Graphs
Slicers (Interactive Filters)
Data Cleaning

📂 Dataset Summary
Time Period: Year 2022
Data Includes:
- Order ID
- Gender
- Age Group
- Sales Amount
- Order Status
- Sales Channel
- State

⚙️ Data Process
1️⃣ Data Cleaning
Removed duplicate entries
Handled missing values
Standardized column names
2️⃣ Data Transformation
Created age groups (Teen, Adult, Senior)
Categorized order status
Structured data for analysis
3️⃣ Dashboard Creation
Built interactive dashboard using slicers
Visualized data using bar charts, pie charts, and line graphs

📊 Dashboard Highlights
📌 Key Metrics
Women contribute ~51% of total sales
93% orders successfully delivered
Adults are the highest contributing segment

📈 Visual Insights
Top Channels: Myntra & Flipkart dominate sales
Top State: Maharashtra generates highest revenue
Trend: Seasonal fluctuations observed in monthly sales

💡 Strategic Recommendations
Target Women Customers: Increase product variety and marketing
Channel Optimization: Focus on Myntra & Flipkart
Regional Expansion: Improve reach in low-performing states
Seasonal Strategy: Boost marketing during low-sales months

🚀 Conclusion
This project demonstrates how Excel dashboards can convert raw data into actionable insights. It highlights key business trends and helps in making informed decisions to improve sales performance.
